## まとめ

- **Emmet 記法**を使用し、コーディング時間を短縮しよう
- VSCode ショートカットキーを活用し、コーディング時間を短縮しよう
- **インラインブラウジング**を利用し、スタイルを調整しよう
- style タグ内に CSS を記載すると**可読性が悪くなる**為、CSS は別ファイルに記載しよう
