# タグについて学ぼう

## タグとは

- 開始タグ<> と 終了タグ</>で構成されたマークアップ

---

# コンテンツとは

- 開始タグと終了タグで囲まれた内容

```
<h1>はじめてのHTML</h1>
```

---

# 要素とは

- 開始タグ + 終了タグ + コンテンツ

```
<h1>はじめてのHTML</h1> //←これ全体
```

## 確認方法

### STEP1

html の body 内に記載

### STEP2

ブラウザをリロード(Ctrl + R)
※F5 でもリフレッシュ出来ますが、キャッシュが残る可能性があるので、リロードの癖を付けておくことを強く推奨します

---

## 確認方法

### STEP3

ブラウザに反映されていることを確認

### まとめ

実装 → リロード → 確認
